Installing this NEdit Macro WILL REMOVE ALL YOUR PREVIOUS SETTINGS.
So you might want to start with:
1) close all NEdit windows and then:
2) mv ~/.nedit ~/.nedit.save

Installation:
1) tar xcvf dot_nedit.tar.gz
2) mv dot_nedit .nedit

Optional:
3) cp .nedit/macro-kit/install/dot_Xdefaults ~/.Xdefaults
